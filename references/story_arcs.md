# Story Arcs for App Screenshots

## 1. The "Feature Dive" (Standard Utility)
Best for: Productivity, Tools, Utilities.
*   **Screen 1**: The "Hero" Shot. High-level value prop + Main App Interface.
*   **Screen 2**: Key Feature A (e.g., "Scan Documents"). Zoom in on the action.
*   **Screen 3**: Key Feature B (e.g., "Extract Text"). Show the result.
*   **Screen 4**: Integration / Settings / Variety.
*   **Screen 5**: Social Proof + Call to Action (e.g., "Join 10M+ Users").

## 2. The "Lifestyle Flow" (Consumer Apps)
Best for: Fitness, Dating, Travel, Social.
*   **Screen 1**: The "Dream". Emotional imagery (happy person) + App Hook.
*   **Screen 2**: The "Problem Solver". How the app fits into daily life.
*   **Screen 3**: The "Data" (Quantified Self). Charts, streaks, progress.
*   **Screen 4**: The "Community". Social features, chat, maps.
*   **Screen 5**: "Start Your Journey". Trial offer or final hook.

## 3. The "Game Hype" (Mobile Games)
Best for: Games.
*   **Screen 1**: Key Art / Character close-up. "Epic Battles".
*   **Screen 2**: Gameplay Action (UI Breakout). "Real-time Combat".
*   **Screen 3**: Progression / Loot. "Collect Legendary Gear".
*   **Screen 4**: Multiplayer / Social. "Guild Wars".
*   **Screen 5**: "Play Now".

## 4. The "AI Magic" (GenAI Apps)
Best for: Chatbots, Image Generators, Writers.
*   **Screen 1**: Input -> Output in one shot. "Turn Words into Art".
*   **Screen 2**: Variety of outputs. "Infinite Styles".
*   **Screen 3**: Advanced Control. "Edit with Magic".
*   **Screen 4**: Use Cases. "For Work, Play, & Study".
